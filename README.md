## Project Overview

This repository contains the documentation, code, and dashboard components developed for my Master’s thesis.

The goal of this research is to build a machine learning prediction system and improve its interpretability using explainable AI techniques (e.g., SHAP) and an interactive Power BI dashboard. The system will be evaluated to determine whether interactive visualization improves understanding, trust, and decision quality compared to static explanations.

Dataset → Machine Learning Model → Explainability (SHAP) → Power BI Dashboard → User Evaluation
